GOATOTB — 20 % Discount Code Setup

Step-by-step to attach a 20 % discount offer (code GOATOTB) to ChessEver's existing Premium subscription products on iOS and Android. No new products are created — the offer hangs off the subscriptions you already sell.

Step 1 — Compute the discounted price

You don't enter "20 %" anywhere. Both stores ask for an absolute price. Calculate it once, use it on both:

discounted_price = regular_price × 0.80
ProductRegular priceEnter as discounted price
Premium Annual$ ____.____$ ____.____  (× 0.80)
Premium Monthly (if applicable)$ ____.____$ ____.____  (× 0.80)

Repeat for every storefront/territory price tier you support. Apple and Google will offer the equivalents in local currencies automatically.

iOS Step 2 — App Store Connect

  1. Sign in to appstoreconnect.apple.com → Apps → ChessEver.
  2. Sidebar → Subscriptions → click Premium Annual (or whichever subscription this code covers).
  3. Scroll to Subscription Offer Codes → click Create Offer Code.
  4. Fill in:
    • Reference name: GOATOTB
    • Eligibility: tick New customers and Existing customers (expired)
    • Offer type: Pay as you go
    • Duration: 1 period (= first year if annual, first month if monthly)
    • Price: the discounted price you calculated in Step 1
  5. Add territories (typically All available).
  6. Choose Custom code → type GOATOTB in caps → set a redemption cap (e.g. 10 000) → save.
  7. Repeat steps 2–6 for Premium Monthly if you want the code to work on monthly too. Use the same string GOATOTB — Apple keys offers per-product and the redemption sheet picks the right one based on what the user is buying.

Android Step 3 — Google Play Console

  1. Sign in to play.google.com/console → pick ChessEver.
  2. Sidebar → Monetize with Play → Subscriptions → click Premium Annual's base plan.
  3. Click Add offer.
  4. Fill in:
    • Offer ID: goatotb-20off
    • Eligibility: Developer-determined (this lets a promo code unlock the offer)
    • Phase: add one phase
      • Type: regular phase
      • Duration: 1 billing cycle
      • Price: the discounted price from Step 1
    Save the offer.
  5. Sidebar → Promo codes → Create promo code.
    • Promotion type: Custom code (vanity)
    • Code: GOATOTB
    • Applicable products: tick the offer you just created (goatotb-20off)
    • Max redemptions: e.g. 10 000
    • Start / end dates: set as you like
    Create.
  6. Repeat steps 3–5 for Premium Monthly if you want the code to work on monthly too. Same code string, separate offer per base plan.

Step 4 — Verify

  1. On a real device, sign into a different Apple ID / Google account than your dev one.
  2. Open ChessEver → paywall → Have a code? → enter GOATOTB.
  3. You should see the discounted price in the store's confirmation sheet (e.g. "First year for $X.XX, then $Y.YY/year").
  4. Confirm the redeem. ChessEver should unlock Premium and play the celebration animation when the sheet closes.
What users will see when they redeem

First billing cycle at the discounted price (your 80 % number), then auto-renews at the regular price thereafter. The discount applies once — the next renewal is full price. This is how both stores express "% off" subscription codes.
